# Template for Creating Python Modules
```
module-template/
├── LICENSE
├── pyproject.toml
├── README.md
├── src/
│   └── package_YOUR_USERNAME_HERE/
│       ├── __init__.py
│       └── example.py
└── tests/
```
See the following for instructions: https://packaging.python.org/en/latest/tutorials/packaging-projects/
